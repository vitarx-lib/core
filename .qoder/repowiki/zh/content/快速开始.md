## 简介

Vitarx是一个现代化的前端框架，它融合了React的JSX语法和Vue的响应式系统，旨在提供简洁、高效的开发体验。本指南将指导您如何快速搭建并运行您的第一个Vitarx应用，从安装包到创建组件，再到挂载应用的完整流程。

## 安装vitarx包

要开始使用Vitarx，首先需要通过npm或其他包管理工具安装vitarx包。以下是使用不同包管理器的安装命令：

```bash
# 使用 npm
npm install vitarx

# 使用 yarn
yarn add vitarx

# 使用 pnpm
pnpm add vitarx
```

安装完成后，您就可以在项目中导入和使用Vitarx提供的各种功能。

## 创建HTML宿主文件

创建一个基础的HTML文件作为应用的宿主。这个文件将包含一个用于挂载Vitarx应用的DOM元素。以下是一个简单的HTML宿主文件示例：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>我的Vitarx应用</title>
</head>
<body>
  <div id="app"></div>
</body>
</html>
```

在这个文件中，`<div id="app"></div>` 是应用的挂载点，Vitarx应用将被渲染到这个元素内部。

## 编写根组件代码

使用JSX语法编写根组件代码。根组件是应用的入口点，通常包含一些基本的UI元素和交互逻辑。以下是一个使用ref和onMounted的计数器示例：

```tsx
import { createApp, ref, onMounted } from 'vitarx'

function App() {
  const count = ref(0)
  
  const increment = () => {
    count.value++
  }
  
  onMounted(() => {
    console.log('组件已挂载')
  })
  
  return (
    <div>
      <h1>计数器</h1>
      <p>当前计数: {count.value}</p>
      <button onClick={increment}>增加</button>
    </div>
  )
}
```

在这个示例中，`ref`用于创建一个响应式的计数器变量，`onMounted`用于在组件挂载后执行一些初始化操作。

## 创建和挂载应用实例

使用`createApp`函数创建应用实例，并调用`mount`方法将其挂载到DOM元素上。这是启动Vitarx应用的关键步骤：

```typescript
import { createApp } from 'vitarx'
import App from './App'

// 创建应用实例
const app = createApp(App)

// 挂载到DOM元素
app.mount('#app')
```

`createApp`函数接收根组件作为参数，返回一个应用实例。`mount`方法则将应用实例挂载到指定的DOM元素上，使其可见于页面。
